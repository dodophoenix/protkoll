## Warum

Du hast ein paar Arzttermine wahrgenommen, ein paar Symptome bemerkt, und willst eigentlich nur eine simple Frage beantworten können: **Wie lange war eigentlich mal wieder alles gut?**

Dafür einen Account anlegen, ein Abo abschließen, seine Gesundheitsdaten in irgendeine fremde Cloud hochladen — für so etwas Simples? Nein. Es gibt genug überteuerte Gesundheits-Apps, die genau dafür monatlich kassieren und trotzdem nicht das tun, was man eigentlich braucht.

**Protokoll** macht nichts davon. Es ist eine einzige Seite, die sich wie eine App installieren lässt, komplett offline läuft, und deren Daten dir gehören — sonst niemandem.

## Was ist das

Protokoll ist eine kleine, installierbare Web-App (iPhone, Android, Desktop) für:
- **Symptome, Arztbesuche, Diagnostik und Sonstiges** — vier Kategorien, jede mit eigener Farbe
- **Freitext-Notizen mit Markdown** — fett, kursiv, Listen, kurze Überschriften
- **PDF-Befunde als Anhang** bei Diagnostik-Einträgen, direkt in der App gespeichert und wieder herunterladbar
- **Arzt-Stammdaten** mit antippbarer Telefonnummer (ein Tap → Anruf)
- **Die Zeiträume dazwischen** — die Kernfrage überhaupt: Wie viele Tage war es symptomfrei?

## Features

| Bereich | Was es tut |
|---|---|
| 📊 **Dashboard** | Kennzahlen auf einen Blick: Tage seit letztem Eintrag, längste beschwerdefreie Phase, Kategorien-Verteilung |
| 🧵 **Verlauf** | Ein Protokoll-Strip mit farbigen Knoten pro Kategorie, dazwischen die Lücken in Tagen — dein persönliches "alles war gut"-Barometer |
| 🔍 **Filter** | Toggle-Filter nach Kategorie, sowohl im Dashboard als auch im Verlauf |
| 📅 **Kalender-Export** | Jeder Eintrag lässt sich als Termin (ICS) teilen oder direkt in Google Kalender öffnen |
| 📎 **PDF-Anhänge** | Befunde bei Diagnostik-Einträgen hochladen, umbenennen, herunterladen, löschen |
| 🖨️ **Druckansicht** | Kompakte A4-Aufbereitung des gesamten Verlaufs, ideal zum Mitbringen zum Arzttermin |
| 💾 **Export / Import** | Der komplette Datenbestand (inkl. PDFs) als Datei — für Backups oder den Wechsel auf ein neues Gerät |
| ⚙️ **Einstellungen** | Steuerbare Dashboard-Ansicht, Datenverwaltung, Update-Check |

## Installation

1. **[dodophoenix.github.io/protkoll](https://dodophoenix.github.io/protkoll/)** öffnen (Safari auf iPhone/iPad, Chrome auf Android)
2. Teilen-Menü → **„Zum Home-Bildschirm"**
3. Fertig — ab jetzt ein eigenes App-Icon, komplett offline nutzbar

## Privatsphäre

Deine Daten bleiben auf deinem Gerät, im sogenannten Local Storage deines Browsers. Es gibt keinen Server, keine Cloud, keinen Account, kein Tracking — niemand außer dir sieht jemals etwas davon. Was du einträgst, gehört dir und sonst niemandem, bis du selbst entscheidest, es zu exportieren.

## Für wen ist das nicht gedacht

Protokoll ersetzt keine ärztliche Beratung und ist kein Medizinprodukt. Es ist ein persönliches Notizwerkzeug, kein Diagnosetool. Bei akuten Beschwerden immer ärztlichen Rat einholen.
